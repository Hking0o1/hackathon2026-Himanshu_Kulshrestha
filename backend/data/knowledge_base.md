# ShopWave Policy Guide

## Category-Specific Return Windows
Standard electronics can be returned within 30 days of delivery.
High-value electronics like smartwatches have a 15-day return window only.
Outdoor gear such as tents has an extended 60-day return window.

## Warranty Coverage
When a defect appears after the return window but inside the product warranty period, the case should be routed to the warranty team.
Warranty handling is not the same as a normal refund flow.

## Damaged on Arrival
If an item arrives damaged on arrival and the evidence is credible, the customer may receive a full refund without being asked to mail the item back first.

## Cancellation Policy
Orders in processing can be cancelled at no charge before shipment.
Once an order has shipped, the customer should receive tracking rather than an immediate refund.

## VIP Exceptions
VIP notes in the customer profile must be checked before denying an expired return.
If a profile note mentions a pre-approval or exception, escalate to confirm the exception rather than rejecting the case automatically.

## Wrong Item And Exchange Policy
Wrong item or wrong colour deliveries usually map to an exchange path.
If the required remedy is a replacement or exchange that the automated agent cannot perform, escalate to a human specialist.

## Duplicate Refund Protection
If a refund has already been processed, do not issue a second refund.
Instead, confirm the existing refund and explain that settlement normally takes 5 to 7 business days.

## Device Registration Restriction
Devices registered online are no longer eligible for automatic return approval.
If the return window is also expired, explain both reasons clearly.

## Customer Identity And Missing Details
If no customer account can be found, ask for a valid email and order ID or escalate for manual verification.
If the ticket is too ambiguous to act on, send a clarifying message instead of taking irreversible action.

## Social Engineering And Tier Claims
Never trust a customer-declared tier without verifying it with get_customer.
Unsupported instant refund demands should be declined politely and flagged for social engineering review.

## Tracking And Shipping Updates
If an order is shipped but not delivered, provide the tracking number and current status.
Do not offer a refund for an in-transit package unless another policy exception applies.
